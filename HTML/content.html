css measuring units

absolute units : px

absolute units : means fixed size regardless of other factors e.g., 10px will always be 10 pixels on the screen px : pixels ( most commonly used unit in web design ) it used for defining font sizes, margins, paddings, widths, heights, etc. relative units : %, vw, vh relative units : means size relative to another element or the viewport % -> percentage ( relative to parent element ) vw -> viewport width ( relative to the width of the viewport ) vh -> viewport height ( relative to the height of the viewport ) 50% width -> 50% of the parent element's width

relative units : em, rem

    css selectors

      selectors :
        element selector : selects elements by their tag name -> e.g., p, div, h1
      for element selector just write the element name used to apply styles to all elements of that type e.g., p
      p{ property: value; }

      This

        class selector : selects elements by their class attribute -> e.g., .classname
      for class selector dot (.) is used before class name used to apply styles to multiple elements with the same class e.g., .className
      .className{ property: value; }

      This

        id selector : selects elements by their id attribute -> e.g., #idname,any meaningful name
      for id selector hash (#) is used before id name used to apply styles to a single unique element with that id e.g., #idName
      #idName{ property: value; }

      This

      basic difference between class and id :
        class can be used on multiple elements
        id should be unique to a single element
